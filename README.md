# Балабанов Александр БАС-1 Лабораторная 4
**Задание 1**

**Задача 1**

**Текст задачи**

Разработать класс с тремя целочисленными полями. Создать конструктор копирования.
Разработать метод, вычисляющий максимальную из последних цифр полей. Перегрузить метод
ToString() для формирования строки из полей класса. Реализовать дочерний класс (его
содержание предложить самостоятельно и описать в решении: какой содержательный смысл
имеют поля; реализовать конструкторы; предложить и реализовать 2-3 метода). Протестировать
все конструкторы и другие методы базового и дочернего классов.
**Алгоритм решения**

Программа состоит из трех модулей, работающих в иерархии наследования. Базовый класс TripleNumber хранит три целых числа A, B, C с конструкторами по умолчанию (5,5,5), с параметрами и копирования, предоставляет метод MaxLastDigit для нахождения максимальной последней цифры среди модулей чисел и переопределенный ToString. Производный класс TriangleSides наследует TripleNumber, добавляя валидацию сторон треугольника: через ключевое слово new переопределяет свойства A, B, C с проверкой на положительность, имеет метод IsValidTriangle, проверяющий неравенство треугольника, GetPerimeter для вычисления периметра и GetTriangleType для определения типа (равносторонний, равнобедренный, разносторонний), а в конструкторе при получении отрицательных сторон заменяет их на (5,5,5) с предупреждением. Класс Program содержит методы ReadInt и ReadPositiveInt для безопасного ввода, InputTripleNumber для ручного создания TripleNumber, InputTriangleSides для ручного ввода сторон с проверкой неравенства треугольника, а в методе Main последовательно тестирует конструкторы обоих классов, метод MaxLastDigit, создание треугольника со сторонами (3,4,5) и некорректными значениями, изменение свойств с контролем отрицательных значений, ручной ввод чисел и сторон треугольника с выводом периметра и типа.

**Задание 2**

**Задача 1**

**Текст задачи**

Определить, попадает ли заданное число в отрезок. Результат должен быть типа bool. 
Унарные операции:
! - вычислить длину отрезка, результат должен быть типа double;
++ увеличить координаты границ отрезка на 1.
Операции приведения типа:
int (явная) – результатом является целая часть координаты х;
double (неявная) – результатом является координата y.
Бинарные операции:
+int d, целое число (лево- и правосторонняя операция, координаты границ
увеличиваются на число d);
< LineSegment, целое число – результат равен true, если целое число
попадает в заданный отрезок и false–в противном случае.


**Алгоритм решения**

Программа состоит из двух модулей: класс LineSegment, который определяет отрезок с координатами точек, хранит их в полях, предоставляет три конструктора (по умолчанию, с параметрами и копирования), метод Contains для проверки попадания числа в проекцию отрезка на ось X, а также перегруженные операторы — унарные (! для вычисления длины и ++ для сдвига отрезка), явное приведение к int (целая часть X1), неявное к double (значение Y1), бинарный оператор + (сдвиг отрезка на целое число с обеих сторон) и операторы сравнения < и >, которые фактически вызывают метод Contains; и класс Program, который в методе Main последовательно тестирует все возможности класса LineSegment через создание объектов, вывод результатов на консоль и ручной ввод координат отрезка пользователем, после чего демонстрирует вычисление длины, приведения типов и проверку попадания числа.
